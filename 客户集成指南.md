# 📋 给客户的集成指南

## 🎯 一行代码集成客服系统

亲爱的客户，恭喜您选择我们的客服系统！以下是超简单的集成方法：

### 🚀 方法一：零配置集成（最推荐）

**只需要在您网站的 `</body>` 标签前添加这一行代码：**

```html
<script src="https://您的域名.com/simple-customer-service-sdk.js"></script>
```

**就这么简单！** 我们的SDK会自动：
- ✅ 找到您现有的客服按钮
- ✅ 绑定新的聊天功能
- ✅ 创建现代化的聊天界面
- ✅ 连接到我们的服务器

### 🛠️ 方法二：自定义配置

如果您想自定义欢迎消息等设置：

```html
<script>
window.CUSTOMER_SERVICE_CONFIG = {
    welcomeMessage: '您好！欢迎咨询我们的客服',
    ui: {
        title: '24小时在线客服',
        placeholder: '请描述您的问题...'
    }
};
</script>
<script src="https://您的域名.com/simple-customer-service-sdk.js"></script>
```

### 📱 完全兼容您现有的网站

**无需担心兼容性问题！** 我们的SDK智能识别以下客服按钮：
- `#cb` (您网站的客服按钮)
- `.customer-service`
- `.kefu`
- `.service-btn`
- 以及其他常见的客服按钮

- 🔧 **样式隔离**：不会影响您网站原有的样式
