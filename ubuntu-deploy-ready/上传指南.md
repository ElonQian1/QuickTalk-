## 💾 上传方法

### 方法1: SCP 上传 (推荐)

```bash

```

### 方法2: 压缩上传

```bash
# 1. 在Windows上压缩
tar -czf ubuntu-deploy.tar.gz ubuntu-deploy-ready

# 2. 上传压缩包
scp ubuntu-deploy.tar.gz root@43.139.82.12:/root/

# 3. 在Ubuntu服务器上解压
ssh root@43.139.82.12
cd /root
tar -xzf ubuntu-deploy.tar.gz
```
