🚀 最简单的第三方客服系统集成方案

💡 这就是您朋友最容易接受的集成方式！
只需要添加一行 script 标签，自动绑定现有客服按钮。

方式一：一行代码集成（最推荐）

您朋友只需要在网站的 </body> 标签前添加：

<script src="https://您的域名.com/simple-customer-service-sdk.js"></script>

就这么简单！ SDK会自动：

  • ✅ 查找并绑定现有的客服按钮（#cb）
  • ✅ 创建现代化的聊天界面
  • ✅ 连接到您的服务器
  • ✅ 处理所有的通信逻辑

方式二：带配置的集成

如果需要自定义配置：

<script>
window.CUSTOMER_SERVICE_CONFIG = {
    apiUrl: 'https://您的域名.com/api',
    wsUrl: 'wss://您的域名.com/ws',
    welcomeMessage: '您好！我是您的专属客服',
    ui: {
        title: '24小时在线客服',
        placeholder: '请描述您的问题...'
    }
};
</script>
<script src="https://您的域名.com/simple-customer-service-sdk.js"></script>

方式三：手动初始化（最灵活）

<script src="https://您的域名.com/simple-customer-service-sdk.js"></script>
<script>
// 手动创建客服实例
const customerService = new SimpleCustomerService({
    wsUrl: 'wss://您的域名.com/ws',
    theme: 'blue'
});

// 手动绑定任意按钮
document.querySelector('#my-custom-btn').onclick = function() {
    customerService.toggle();
};
</script>

🎯 为什么这是最简单的方案？

对您朋友的好处：

  • 零修改： 不需要改动现有代码
  • 零配置： 默认就能工作
  • 零依赖： 不需要额外的库
  • 自动兼容： 自动查找并绑定客服按钮
  • 响应式： 自动适配手机端

对您的好处：

  • 标准化： 一套代码适用所有网站
  • 易维护： 集中式管理和更新
  • 易推广： 集成门槛极低
  • 易扩展： 可以添加更多功能

📋 开发步骤

第1步：部署SDK文件

将 simple-customer-service-sdk.js 部署到您的服务器，确保可以通过URL访问。

第2步：修改WebSocket服务器

确保您的 WebSocket 服务器支持SDK的消息格式。

第3步：测试集成

在测试页面添加SDK，验证功能正常。

第4步：提供给客户

给您朋友一行集成代码，他粘贴到网站就能用。

⚠️ 重要提醒

  • 确保您的服务器支持HTTPS（生产环境必须）
  • 配置正确的CORS策略
  • 测试WebSocket连接稳定性
  • 提供备用的HTTP轮询方案
